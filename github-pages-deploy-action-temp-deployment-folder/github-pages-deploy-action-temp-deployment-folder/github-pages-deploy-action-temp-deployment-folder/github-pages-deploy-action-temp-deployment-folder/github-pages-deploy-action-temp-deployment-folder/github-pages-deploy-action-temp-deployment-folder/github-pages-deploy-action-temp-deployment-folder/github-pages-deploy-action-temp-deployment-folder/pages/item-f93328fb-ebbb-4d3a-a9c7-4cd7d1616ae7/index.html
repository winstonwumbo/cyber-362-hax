Please review the following documentation for installing and configuring Python, Anaconda, and VS Code on your macOS device.

Install Python

  1. Download Python from the official site: python.org
  2. Select Download macOS 64-bit universal installer
  3. Select python-3.13.7-macos11.pkg from the Downloads folder in Finder
  4. Complete the Install Python dialogue with default options
  5. Visit the /Applications/Python 3.13/ folder in Finder
  6. Run the script file Install Certificates.command
  7. Open Terminal
  8. Test Python by entering python --version
  9. If the version number appears, then Python has been successfully installed!

Install Conda

  1. Visit the official Anaconda website: anaconda.com, select Skip Registration
  2. Select 64-Bit (Apple silicon) Graphical Installer  or 64-Bite (Intel chip) Graphical Installer under Mac - Python 3.13, the correct option depends on your MacBook model
  3. Select Miniconda3-latest-MACOSX-<model type>.pkg from your Downloads folder iin Finder
  4. Complete the Install Miniconda3 20205.06-1 dialogue with default options
  5. Open Terminal
  6. Test Conda by entering conda --version
  7. If the version number appears, then Conda has been successfully installed!

Install and Configure VS Code

  1. Visit the official Visual Studio Code website: code.visualstudio.com
  2. Visit the Downloads page, select Mac - macOS 11.0+
  3. Select VSCode-darwin-universal.zip from your Downloads folder in Finder
  4. When the ZIP file uncompresses, move Visual Studio Code.app to your Applications folder
  5. Open VS Code by double-clicking the .app file, then pin VS Code to your dock
  6. In VS Code, open the Command Palette (Cmd+Shift+P)
  7. Enter shell command, then select Shell Command: Install 'code' command in PATH
  8. Select the Extensions icon on the left sidebar, then search for Python in the Extensions Marketplace
  9. Install the Python extension pack
  10. Search for Jupyter in the Extensions Marketplace, then install the Jupyter extension pack

Test Jupyter Notebooks

  1. Create a CYBER 362 folder to store assignments in Finder (Suggestion: Documents/Cyber362), open VS Code in this folder
  2. Create a temporary file named test.py in this folder, enter the text print("Hello World!") and save the file
  3. Select the Run Python File button
  4. If everything was installed successfully, a Hello World! message will appear in your terminal!
  5. Now let's test Jupyter with Conda! Close your current terminal window, then select the Python 3.13 button at the bottom right of VS Code
  6. Select the base (3.13.5) Conda environment, this will change the Python Interpreter that VS Code is using
  7. Select Terminal > New Terminal, then verify that VS Code automatically runs conda activate base in the new terminal window
  8. Enter conda install jupyter to download all the necessary Python packages for using Jupyter Notebooks
  9. Enter conda install numpy matplotlib pandas to download additional Python packages for future assignments
  10. Create another temporary file named test.ipynb, this is the file extension for Jupyter Notebooks
  11. Select the Select Kernel button on the right side of your notebook, then select Python Environments
  12. Select base (Python 3.13.5) Conda Env
  13. Select the + Code button, then enter print("Hello World!") into the Jupyter Notebook cell
  14. If everything was successfully installed, the message Hello World! will appear below!
