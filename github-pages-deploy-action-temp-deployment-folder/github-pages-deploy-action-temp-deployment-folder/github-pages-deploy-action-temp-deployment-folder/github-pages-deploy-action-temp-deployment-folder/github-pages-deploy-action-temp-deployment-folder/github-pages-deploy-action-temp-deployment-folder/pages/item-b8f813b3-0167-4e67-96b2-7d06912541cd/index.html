What is HAX?

A radically simple approach to web authoring and content ownership. The goal of HAX is simple: to help people create rich and engaging web content without needing to understand web code.

HAX is an open-source ecosystem built right here at Penn State! HAX is built with vanilla JavaScript and LitElement on the Web Component standard. The idea is online content should always be accessible, even as third-party frameworks age and diverge.

This thriving community is led by Project Lead Bryan Ollendyke and Faculty Fellow Dr. David Fusco. HAX involves a variety of fields including programming (full stack), UI/UX design, Data Sciences/AI, learning design, education technology development, and more.

Getting Involved?

HAX is always looking for more bright minds, eager to gain valuable skills and to contribute towards a greater community. There are two main organizations at Penn State University Park, HAX Lab and HAX The Club.

  • HAX Lab: A research group focused on open and accessible web technologies. HAX Lab is a collaboration between IST, Arts and Architecture and other partners to empower educational technology and student success through the HAX ecosystem. The web is for everyone, regardless of background!
  • HAX The Club: A student-led organization of enthusiasts connected to HAX Lab. HAX The Club is open to anyone interested in making a mark with open-source! Weekly meetings are on Thursdays at Borland Room 113!

Learn more at hax.psu.edu and github.com/haxtheweb!
