Please review the following documentation for installing and configuring the GitHub Copilot coding assistant to your VS Code application.

Install GitHub Copilot

  1. Open Visual Studio Code from the Start Menu
  2. Select the Extensions icon on the left sidebar, then search for GitHub Copilot in the Extensions Marketplace
  3. Install the GitHub Copilot extension pack
  4. Select the Copilot icon on the bottom status bar, then select Set up Copilot
  5. Select Continue with GitHub as the sign-in method, then enter your GitHub account credentials
    • If you do not have a GitHub account, please create one at: github.com
  6. VS Code will automatically open a new Copilot Chat menu on the right side of the screen. GitHub Copilot is now successfully installed!

Unlock Copilot Pro

The base version of GitHub Copilot severely limits the usage rate, available models, and experimental features for users. Luckily, college students are granted free access to a Copilot Pro subscription through the GitHub Student Developer Pack program.

  1. Visit the official GitHub Education website: education.github.com/pack
  2. Follow the registration steps from GitHub Education, you will be asked to submit an image of your physical Penn State ID
  3. Wait a few days as the GitHub Education application is reviewed
  4. If you receive a Welcome to GitHub Education email, then you have successfully been granted the GitHub Student Developer Pack
  5. Visit the GitHub Education website again, there will now be a long list of affiliated benefits
  6. Select Get direct access on the website GitHub Copilot under All Offers > About GitHub Copilot
  7. Follow the activation steps from GitHub Copilot
  8. If you used the same account to login on VS Code, then Copilot Pro is now successfully activated!
