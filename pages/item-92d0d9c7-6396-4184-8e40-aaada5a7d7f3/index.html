Please review the following documentation for installing and configuring Python, Anaconda, and VS Code on your Windows 10/11 device.

Install Python

  1. Visit the official Python Foundation website: python.org
  2. Select Download Windows installer (64-bit)
  3. Select python-3.13.7-amd64.exe from your Downloads folder in File Explorer
  4. Important: Select the checkbox for Add python.exe to PATH, this allows your terminal to use the standard python command
  5. Select Install Now
  6. Open either Command Prompt or PowerShell from the Start Menu
  7. Test Python by entering python --version
  8. If the version number appears, then Python has been successfully installed!

Install Conda

  1. Visit the official Anaconda website: anaconda.com, select Skip Registration
  2. Select 64-Bit Graphical Installer under Windows - Python 3.13
  3. Select Miniconda3-latest-Windows-x86_64.exe from your Downloads folder in File Explorer
  4. Select Just Me (recommended) during the Select Installation Type step
  5. Complete the remaining Miniconda Installer steps with default options
  6. Open Anaconda Prompt from the Start Menu
  7. Test Conda by entering conda --version
  8. If the version number appears, then Conda has been successfully installed!

Install and Configure VS Code

  1. Visit the official Visual Studio Code website: code.visualstudio.com
  2. Select Download for Windows
  3. Select VSCodeUserSetup-x64-1.103.2.exe from your Downloads folder in File Explorer
  4. Important: Select the checkbox for Add "Open with Code" action to Windows Explorer directory context menu, this allows you to open folders in VS Code directly from File Explorer
  5. Complete the remaining VS Code Setup steps with default options
  6. Open Visual Studio Code from the Start Menu
  7. Select the Extensions icon on the left sidebar, then search for Python in the Extensions Marketplace
  8. Install the Python extension pack
  9. Search for Jupyter in the Extensions Marketplace, then install the Jupyter extension pack

Test Jupyter Notebooks

  1. Open PowerShell with the Run as Administrator option
  2. Enter the command Set-ExecutionPolicy -Scope CurrentUser RemoteSigned, this allows VS Code to use Conda environments
  3. Create a CYBER 362 folder to store assignments in File Explorer (Suggestion: Documents/Cyber362), open VS Code in this folder
  4. Create a temporary file named test.py in this folder, enter the text print("Hello World!") and save the file
  5. Select the Run Python File button
  6. If everything was installed successfully, a Hello World! message will appear in your terminal!
  7. Now let's test Jupyter with Conda! Close your current terminal window, then select the Python 3.13 (64-bit) button at the bottom right of VS Code
  8. Select the base (3.13.5) Conda environment, this will change the Python Interpreter that VS Code is using
  9. Select Terminal > New Terminal, then confirm that conda activate base appears in the new terminal window
  10. Enter conda install jupyter to download all the necessary Python packages for using Jupyter Notebooks
  11. Create another temporary file named test.ipynb, this is the file extension for Jupyter Notebooks
  12. Select the Select Kernel button on the right side of your notebook, then select Python Environments
  13. Select base (Python 3.13.5) Conda Env
  14. Select the + Code button, then enter print("Hello World!") into the Jupyter Notebook cell
  15. If everything was successfully installed, the message Hello World! will appear below!
