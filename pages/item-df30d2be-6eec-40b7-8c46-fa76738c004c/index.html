Please review the following documentation to access and learn about the Splunk SIEM platform at Penn State. This will assist you in preparing for Lab 6.

How to Access Splunk

The College of IST hosts its own instance of Splunk Enterprise at the address splunk.ist.psu.edu. However, this IST instance is restricted and can only be accessed from the Penn State networks psu or eduroam to authenticate. Students may simply visit splunk.ist.psu.edu from the computer labs or their own devices while on-campus. 

If you would like to access splunk.ist.psu.edu remotely, please follow one of the two methods below. Students will only need to choose one approach. You do not need the GlobalProtect VPN to use WinLabs, nor vice versa.

1. GlobalProtect VPN

Penn State provides a free service GlobalProtect VPN for remote connection to secured university resources. By locally installing the GlobalProtect VPN client, your network traffic will be encrypted, then safely routed to a Penn State server. Please follow the correct University IT guide for your operating system below:

  • GlobalProtect for Windows
  • GlobalProtect for macOS
  • GlobalProtect for Linux

For more information on GlobalProtect VPN in general, please reference this University IT article:

  • GlobalProtect VPN - Overview

2. WinLabs

The College of IST also hosts a free Virtual Desktop Infrastructure (VDI) solution called WinLabs. A VDI is a virtual machine cluster that provisions a full desktop environment to remote users. WinLabs is a browser-based platform, which students can use from Chrome, Firefox, and other popular browsers.

To access WinLabs, please visit the address winlabs.ist.psu.edu. You will be instructed to login with your Penn State ID and password. (ex: wjw5326)

You will now have access to a remote virtual machine on the Penn State network. For more information, you can reference the following resources:

  • Connecting to WinLabs
  • IST Classroom - WinLabs

Logging into Splunk

Once you are on the Penn State network (from one of the three methods above), please visit the address splunk.ist.psu.edu. The Splunk Login Screen will become visible. You will need to enter your Penn State ID (ex: wjw5326, Do Not Include the @psu.edu) and password.

After submitting the correct login credentials, you will gain access to the Splunk Enterprise Dashboard.
